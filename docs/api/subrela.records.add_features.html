subrela.records.add_features function¶

subrela.records.add_features(s, features)[source]¶

Add features.

Parameters

  • s (pandas.Series) – Score records

  • features (list[str]) – Names of added features.

Returns

pandas.Series – New score records.

Raises

ValueError – If a feature in features already exists.

See also

subrela.records.remove_features()

Examples

>>> # just for the output format
>>> pandas.set_option('display.multi_sparse', False)

>>> s = from_arrays([[True, False, False], [True, False, True]],
...                 [0.2, 0.8],
...                 features=['A', 'B', 'C'])
>>> s
A     B      C
True  False  False    0.2
True  False  True     0.8
dtype: float64
>>> add_features(s, ['X', 'Y'])
A     B      C      X      Y
True  False  False  False  False    0.2
True  False  True   False  False    0.8
dtype: float64
